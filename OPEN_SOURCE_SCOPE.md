# EGT 稳健开源边界（Scope）

## 开源（Public）
- EGT 三层定义（Emotion/Game/Thinking）
- 标准输出模板（结论->动作->代价->下一步）
- 安全门禁（高风险动作二次确认）
- 熔断与复盘模板

## 保留（Private）
- 私域记忆文件与人物档
- 具体身份锚点与关系上下文
- 运营日志、故障细节、内网拓扑
- 账号、token、密钥、支付信息

## 发布前自检
- [ ] 无隐私数据
- [ ] 无凭据/密钥
- [ ] 无可定位个人身份信息
- [ ] 无可复现攻击路径的敏感细节

## 对外说明模板
> 本项目开源的是协作协议与执行方法，不包含任何私有记忆、身份配置或敏感数据。
